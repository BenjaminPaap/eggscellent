Eggscellent Help

Eggscellent is a task manager designed mainly for OS X Mountain Lion. This method uses 25 minute increments for intense focus on a single task, followed by 5 minutes of break time.

We have developed this application to help increase productivity, and have designed the app to be visually appealing while integrating with your other task managers in hopes of helping you accomplish all you set out to do.
